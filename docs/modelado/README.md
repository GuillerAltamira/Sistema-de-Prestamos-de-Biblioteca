# Sistema de Préstamos de Biblioteca

## Descripción
Sistema para gestionar préstamos de libros en una biblioteca académica.

## Reglas de negocio
- Máximo 3 préstamos por estudiante
- No permitir préstamo con multas pendientes
- Validación de disponibilidad del libro
